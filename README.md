# MBEAUTYQUEEN Salon Web Frontend

A clean and user-friendly frontend interface developed for our Mini Capstone client, **MBeautyQueen Brows and Lashes Salon**, located in Brgy. Mahabang Parang, Angono, Rizal.

This project represents the **frontend-only** implementation. Full-stack development will follow in the next semester of our 2nd year.  
The system uses **Local Storage** as its temporary database.
